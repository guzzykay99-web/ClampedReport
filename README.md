# Clampet Report

A comprehensive forensic analysis of the Clampet.E01 image revealing evidence of moonshine production and money laundering.

## Executive One-Sentence Description
This project analyzes the Clampet.E01 disk image using standard forensic tools to uncover evidence of illegal moonshine operations and money laundering.

## Overview
This report documents the forensic examination of the Clampet.E01 hard drive image using Autopsy, FTK Imager, and John the Ripper. The investigation uncovered evidence linking the user, Jethro Clampet, to moonshine production, distribution, and financial laundering. Findings include incriminating documents, suspicious searches, password-protected files, and media indicating an organized criminal operation.

## Tools Used
| Tool | Purpose |
|------|---------|
| Autopsy 4.22.1 | Forensic analysis of the E01 image |
| FTK Imager (Exterro) | Hash verification and image validation |
| John the Ripper (Kali Linux) | Cracking passwords on protected files |
| Notepad | Timeline and notes |
| Snipping Tool | Capturing evidence screenshots |

## Process Summary
1. Validated the Clampet.E01 image using FTK Imager (MD5/SHA1 match).  
2. Conducted an initial sweep in Autopsy and tagged evidence.  
3. Examined files linked to moonshine.  
4. Analyzed files relating to money laundering.  
5. Cracked three password-protected files with John the Ripper.  
6. Compiled all findings into the final report.

## Summary
Evidence recovered from the Clampet.E01 image shows that Jethro Clampet was involved in both producing illegal moonshine and laundering the profits. Incriminating documents, images, and searches all point to active engagement in these activities. Password-protected files contained customer records, financial logs, and coded messages, indicating deliberate criminal organization beneath an ordinary Windows XP system.
